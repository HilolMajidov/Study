# CMakeHello

Простой пример проекта на **C++17** с использованием **CMake** для сборки.

---

##  Структура проекта

CMakeHello/
├── CMakeLists.txt # настройки сборки (CMake)
├── main.cpp # точка входа программы
├── headers.cpp # реализация функций
├── headers.hpp # заголовочный файл
└── build/ # создаётся автоматически при сборке


## Требования

Перед началом убедись, что установлены:

- **CMake** версии **4.1.2** или новее  
- **C++ компилятор** (например, `g++`, `clang`, или `msvc`)

Проверить версии можно так:
``` cmake --version ```
``` g++ --version```
Сборка проекта
1️ Сгенерировать файлы сборки

```cmake -S . -B build```
Эта команда создаёт папку build/ и подготавливает проект к компиляции.

2️ Собрать проект
```cmake --build build```
После этого внутри build/ появится исполняемый файл CMakeHello.exe.

3️ Запустить программу
```.\build\CMakeHello.exe```
Если ты используешь PowerShell, не забудь про .\ перед именем файла.

Полезные команды
Очистить проект	```cmake --build build --target clean```
Пересоздать сборку с нуля	```Remove-Item -Recurse -Force build```
Сборка с оптимизацией	```cmake -S . -B build -DCMAKE_BUILD_TYPE=Release```
